# 🎰 로또번호 추첨기

간단한 웹 기반 로또번호 추첨 프로그램입니다.

## 기능

- 1부터 45까지의 숫자 중 6개를 무작위로 추첨
- 자동 오름차순 정렬
- 애니메이션 효과로 시각적 재미 제공
- 반응형 디자인

## 사용 방법

1. `index.html` 파일을 브라우저로 열기
2. "추첨하기" 버튼 클릭
3. 행운의 번호 확인!

## 기술 스택

- HTML5
- CSS3 (애니메이션, 그라데이션)
- JavaScript (Vanilla JS)

## 실행

```bash
# 브라우저에서 파일 열기
start index.html  # Windows
open index.html   # macOS
xdg-open index.html  # Linux
```

또는 `index.html` 파일을 직접 더블클릭하세요.
